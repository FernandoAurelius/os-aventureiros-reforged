## CurseForge Phase 4 Mods

Copie estes arquivos para a pasta `mods/` da sua instancia do CurseForge:

- `oar_combat-0.1.0-alpha.1.jar`
- `bettercombat-neoforge-2.3.2+1.21.1.jar`
- `spartan_weaponry_unofficial-1.1.0-1.21.1.jar`
- `player-animation-lib-forge-2.0.4+1.21.1.jar`
- `cloth-config-15.0.140-neoforge.jar`

Esta pasta existe para facilitar a distribuicao do pacote de combate da Fase 4 para o cliente.
