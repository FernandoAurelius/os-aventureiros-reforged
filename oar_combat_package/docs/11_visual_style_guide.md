# Visual style guide

## Mythril

Visual desejado: Tolkien-style, elfico, nobre e leve.

- Cor: prata-azulada clara.
- Textura: polida, com brilho sutil, quase lunar.
- Forma: linhas elegantes, runas finas, desenho refinado.
- Evitar: ciano neon sci-fi, cristal bruto, visual plastico.

## Adamantite

Visual desejado: metal ancestral pesado, austero e implacavel.

- Cor: gunmetal escuro, ferro negro, obsidiana metalica.
- Textura: densa, opaca, com brilho roxo/cinza muito contido.
- Forma: placas grossas, geometria monumental, pouca ornamentacao.
- Evitar: parecer Netherite recolorida ou Netherite com entalhes.

## Orichalcum

Visual desejado: material raro, arcano, verde escuro brilhante.

- Cor: verde profundo com brilho esmeralda.
- Textura: metalica, nao slime/cristal comum.
- Forma: lingote com entalhes/runas no proprio metal.
- Uso: fragmentos, ingots e upgrades lendarios.
- Evitar: parecer emerald ingot simples ou item radioativo.

## Bronze / Steel / Silver

- Bronze: cobre quente, utilitario, early game.
- Steel: cinza frio, militar, confiavel.
- Silver: claro, nobre, anti-undead/mistico.
