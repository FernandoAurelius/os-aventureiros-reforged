# Loot modifier candidate

NeoForge Global Loot Modifiers should be used only after confirming the exact serializer/type API for NeoForge 21.1.229.

Recommended behavior:
- inject tin/bronze low-value loot into low-tier targets
- inject steel/silver into mid-tier targets
- inject mythril/adamantite fragments into high-tier targets
- inject orichalcum_fragment with very low weight only into high-danger structures

Do not override entire loot tables from other mods unless absolutely necessary.
