# oar_combat scaffold

Projeto inicial do addon NeoForge para a expansao de combate e armaduras.

Este scaffold prioriza a Fase 1: itens base, blocos, ores, tags, recipes, modelos e texturas placeholder.

## Importante

O codigo deve ser validado/adaptado no ambiente real da VPS com NeoForge 21.1.229. A API exata de ferramentas/armaduras em 1.21.1 deve ser confirmada antes de ativar o codigo candidato em `src_candidate_disabled`.

## Primeiro comando

```bash
./gradlew clean build
```

Se o projeto nao tiver wrapper, gere a partir do MDK/Gradle real usado na VPS ou use a instalacao local do Gradle.
